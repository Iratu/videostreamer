How to stream

First, you need to signup on the service, proceed to Signup page to do this

When you done -- you'll be having one stream in your profile

There you shall see a stream token, which will serve as private key.

Do not disclose it to anyone you don't wish to be streaming on your channel

Then you need to get a broadcaster, capable of video/audio streaming over RTMP protocol.

For *nix systems the best choice in my opinion is the ffmpeg utility

For windows -- it might be OBS (Open Broadcaster Streamer)

Configuring ffmpeg

ffmpeg  -f x11grab -s 1366x768 -i :0.0 -f pulse -ac 2 -i loopback.monitor -vcodec libx264 -tune zerolatency -preset ultrafast -pix_fmt yuv420p -s 1366x768 -ar 44100 -threads 0 -f flv "{{rtmpPrefix}}/live/yourprivatetoken"

Configuring OBS

Broadcast Settings -> Mode: Live Stream
Broadcast Settings -> Streaming Service: Custom
Broadcast Settings -> FMS URL: {{rtmpPrefix}}/live
Broadcast Settings -> Play Path/Stream key: yourprivatekey
Broadcast Settings -> Auto-Reconnect: true
Advanced -> Video -> x264 CPU Prset: Ultrafast
Advanced -> Video -> Custom x264 Encoder Settings: tune=zerolatency
